# react-tsx-responsive-grid

Small Vite + React + TypeScript demo for a responsive CSS grid layout.

## Run locally

From this folder:

```bash
npm install
npm run dev
```

Then open the URL Vite prints (usually `http://localhost:5173`).

## Other scripts

- `npm run build` — production build to `dist/`
- `npm run preview` — serve the built `dist/` locally

## Static HTML demos

This directory also includes `css-grid-demo-*.htm` files you can open in a browser for plain HTML/CSS grid experiments (no dev server required).
