# t76

A simple Python module.

## Installation

```bash
pip install t76
```

## Usage

```python
import t76

# Use the greeting function
print(t76.greet("World"))

# Use the calculator
calc = t76.Calculator()
result = calc.add(5, 3)
print(f"5 + 3 = {result}")
```

## Features

- Simple greeting function
- Basic calculator class
- Easy to use and extend
